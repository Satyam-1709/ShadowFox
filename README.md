📊 ShadowFox Internship — Beginner Task
Python Data Visualization (Matplotlib & Seaborn)
📌 Project Overview

This project is part of the ShadowFox Data Science Internship beginner task. The objective is to explore Python visualization libraries and create different types of charts using Matplotlib and Seaborn. The project demonstrates how data visualization helps in understanding patterns, trends, and distributions in datasets.

🎯 Objectives
Learn basic data visualization concepts
Understand Matplotlib and Seaborn libraries
Create multiple chart types
Compare both visualization libraries
Visualize sample datasets
